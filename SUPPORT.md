# Support

## 关于

Docker MCP Server 是一个基于 Model Context Protocol (MCP) 的 Docker 管理工具，专为 AI Agent 设计，帮助 AI 通过 MCP 协议管理 Docker 容器和服务。

## 功能

- 🔧 容器管理：创建、启动、停止、删除容器
- 📋 容器列表：查看所有容器状态
- 📝 日志查看：获取容器日志
- 🔍 容器检查：查看容器详细信息
- 📦 Compose 支持：管理 docker-compose 服务
- ⚡ 命令执行：在运行中的容器内执行命令
